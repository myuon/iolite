# Iolite

Iolite is a lightweight programming language for creating simple CUI/GUI applications.

**Everything is in development.**

## Compiler Architecture

```
Iolite source code
↓
Lexer
↓
Parser
↓
Typechecker
↓
IR CodeGenerator + Optimizer
↓
VM CodeGenerator
↓
Linker
↓
Bytecode Emitter
↓
VM
```
